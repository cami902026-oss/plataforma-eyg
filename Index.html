🤖 ENERGY IA — URL del Proxy seguro

Pega aquí la URL del proxy donde reside la API Key de Claude. La clave NUNCA se expone al navegador.
✅ Recomendado (gratis): Google Apps Script. Crea un proyecto en script.google.com, pega el código del archivo claude-proxy.gs, en Configuración del proyecto → Propiedades del script agrega CLAUDE_API_KEY con tu sk-ant-..., despliega como Aplicación web (acceso: cualquier usuario) y pega aquí la URL /exec.
Alternativa: URL base de Netlify (requiere plan pago si superas 125k req/mes).

📝 URL Evaluación de Desempeño (Google Form)

URL del Google Form que se abre desde el botón 📝 de cada miembro del equipo.
Si incluyes {NOMBRE} en la URL, ENERGY lo reemplaza por el nombre del evaluado (útil para prellenar campos con ?usp=pp_url&entry.XXX={NOMBRE}).

🧹 Resetear datos locales

Si ves tareas, OPs o proyectos viejos que ya borraste y vuelven a aparecer, este botón limpia tu localStorage y vuelve a descargar todo desde GitHub (la fuente real del equipo).

📤 Proxy GitHub (escritura compartida sin PAT)

Recomendado para todos los dispositivos del equipo. Permite que ENERGY guarde cambios al servidor sin que cada persona tenga que configurar su propio Token GitHub.
El Token vive seguro en Google Apps Script (Propiedades del script). Pega aquí la URL /exec que generas al desplegar el archivo github-proxy.gs.
Esta URL se sincroniza automáticamente con todo el equipo via data/config.json.

🔔 Token GitHub (Notificaciones Teams)

Token personal de GitHub con permiso workflow. Se guarda localmente en tu navegador.
